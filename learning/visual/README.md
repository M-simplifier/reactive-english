# Reactive English Visual Learning

This is a local browser learning resource for understanding the public
Reactive English repository: architecture, stack, assurance-driven development,
authentication, vocabulary learning, deployment, CI/CD, and cost/security
boundaries.

Run from the repository root:

```sh
npm run learning:serve
```

Then open:

```text
http://localhost:8090
```

The generated hero artwork in `assets/generated-system-hero.png` was created as
a project-bound visual asset for this learning resource. The rest of the
diagrams are code-native HTML/CSS/SVG so they stay inspectable in the repo.
